# SMB AI Workflow Audit Kit

A local-only starter kit for a productized small-business AI workflow audit. It helps map repetitive work, score automation candidates, and produce a safe first-action report before touching real accounts or customer data.

This repository currently uses **fictional sample data only**. It is not a client deliverable, not a guarantee of ROI, and not permission to access anyone's tools, inboxes, documents, CRM, accounting system, or customer records.

## What is included

- `samples/fictional-intake.md` — example intake for a fictional local service business.
- `templates/scoring-rubric.md` — workflow scoring model for automation fit and risk.
- `templates/audit-report-template.md` — report structure with required safety/disclaimer sections.
- `scripts/validate.js` — tiny validation check that required guardrails exist in the starter docs.

## Audit flow

1. Confirm scope and written permission before reviewing any real business process.
2. Collect an intake questionnaire using minimal, redacted, non-sensitive examples.
3. List workflows and score them using the rubric.
4. Recommend only boring, reversible, human-approved first automations.
5. Produce a report that includes risks, non-goals, rollback/deactivation steps, and approval gates.

## Guardrails

- Do not collect passwords, API keys, bank details, customer records, payroll data, medical data, or legal case details.
- Do not scrape inboxes, drives, CRMs, accounting tools, or websites without explicit written authorization and a privacy-safe plan.
- Do not auto-send customer messages; draft for human review first.
- Do not promise revenue, savings, compliance, or guaranteed accuracy.
- Prefer read-only discovery, fictional/redacted examples, and reversible automations.
