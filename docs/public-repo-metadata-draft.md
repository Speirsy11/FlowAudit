## Working repo title

SMB AI Workflow Audit Kit

## Short description

Local-first templates for a safety-first small-business AI workflow audit using fictional/sample data.

## Longer description

A local-only starter kit for reviewing and packaging a fixed-scope SMB AI workflow audit. It includes fictional intake and discovery notes, a scoring rubric, a completed sample report, safe intake template, draft-only pilot blueprint, acceptance scorecard, offer/review docs, and approval-gated externalization checks. The kit is designed to help identify practical, reversible, human-reviewed workflow improvements without collecting secrets, customer exports, regulated data, private-system access, outreach, payment setup, or implementation approval.
